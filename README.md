# Module
This module provides quick-access helper functions that print foundational topics in Python programming. It’s designed for beginners, trainers, and students who need a simple, structured reference to core Python concepts.
✅ Features

Overview of Python operators

List of Python data types

Summary of Python data structures

Breakdown of conditional statements

Explanation of loops and loop control statements

Information about functions and types of arguments

Difference between local and global variables

Basic overview of exception handling

Types of programming errors

Categories of modules in Python

✅ Included Functions

Each function prints relevant subtopics:

operators() – arithmetic, logical, relational, assignment, etc.

datatypes() – integer, float, string, boolean, etc.

datastructers() – list, tuple, set, dictionary, frozen set

conditionalstatements() – if, if-else, nested if

loops() – for loop, while loop

loopcontrolstatements() – break, continue, pass

functions() – built-in and user-defined

arguments() – positional, keyword, default, variable-length

variables() – local vs global

exceptionhandling() – try/except/else/finally

errors() – programmer vs runtime errors

modules() – built-in, user-defined, third-party
